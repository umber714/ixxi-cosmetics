
# IXXI Cosmetics

Sitio web ecommerce para cosméticos premium.

## Instalación

1. Clona el repositorio.
2. Instala dependencias: `npm install`
3. Configura variables de entorno: Crea `.env.local` con:
   - `STRIPE_SECRET_KEY=sk_test_...`
   - `STRIPE_WEBHOOK_SECRET=whsec_...`
4. Ejecuta: `npm run dev`

## Despliegue

Despliega en Vercel o Netlify. Configura las variables de entorno en la plataforma.
